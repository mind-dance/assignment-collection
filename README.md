# 学生作业管理系统

## 简介

本项目是数据库大作业（数据库课程设计）。本项目较为综合，包含了数据导入导出（csv和json）、数据库的连接、编写测试所需的测试用例，是本人目前（截止至2024年六一儿童节）最为复杂的项目之一（另一个是CS50x的tideman）。

程序主入口为`app.py`。

有问题可以提issue，有优化可以提PR。

## 技术栈

Python, Flask, SQLite

将数据存放在SQLite中，通过Python进行调用。
